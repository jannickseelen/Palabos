What's with all these beta versions?

Pantheios was forked from a proprietary diagnostic logging library of Synesis Software in 2005.

The original expectation was that it was nearly ready for a 1.0 release, so it started out at beta 2. Since we're now beyond beta 200, it's obvious that that decision was more than a little hasty.

You may reasonably wonder when the final non-beta 1.0 release of Pantheios will eventuate. At the current time - November 2010 - it's expected within the next 2-3 months. And because we're running out of space in the version constant - PANTHEIOS_VER_1_0_1_B203 is 0x010001cb; it cannot go beyond 0x010001ff - there's a strong likelihood that this timetable is realistic.

If you want to contribute to helping Pantheios take off on to a proper release line, please <a href="http://sourceforge.net/projects/pantheios/forums/forum/475313"get in contact.
