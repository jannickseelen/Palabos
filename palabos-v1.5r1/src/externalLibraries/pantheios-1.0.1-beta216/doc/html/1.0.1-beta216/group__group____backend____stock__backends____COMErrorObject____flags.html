#define PANTHEIOS_BE_COMERROROBJECT_F_DONT_OVERWRITE_EXISTING   (0x00100000)

Prevents the Pantheios COM Error Object Stock Back-end from writing to the COM Error Object if there is an existing error object set (for the calling thread).

This is helpful when several levels of code are handling and reporting errors, so as not to lose. the innermost (and most exact) error report.
