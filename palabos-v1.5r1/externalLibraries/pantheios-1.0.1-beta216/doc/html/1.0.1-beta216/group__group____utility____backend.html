Function Documentation

size_t pantheios_util_getCurrentTime ( pan_beutil_time_t *  tm,
int  flags 
)

Gets the current time in a suitable format.

This function gets the current time, and formats it in an operating system-specific manner into the given pan_beutil_time_t instance, using either the local time or the system time (if the flags parameter includes PANTHEIOS_GETCURRENTTIME_F_USE_SYSTEM_TIME).

Specifically, on Windows platforms the function formats according to the current locale settings. On UNIX, the function uses the syslog-standard format (strftimeFmt() format: "%b %d %I:%M:%S") or the pan_beutil_time_t::strftimeFmt member if it is non-NULL.

Parameters:
tmPointer to a pan_beutil_time_t instance, whose str and capacity members must be valid. May not be NULL.
flagsOne or more of the PANTHEIOS_GETCURRENTTIME_F_* flags.
Returns:
The number of characters written into the buffer, or 0 on failure
